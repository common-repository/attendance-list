Simple event attendance list. You can add it to any post or page.

== Description ==

Attendance List is simple plugin to manage multiple events attendance lists, witch you can add one for each post. It also support AJAX requests for subscribing and unsubscribing from lists.

Easy one-click installation.

You can also easy change language by edit language file lang.php located in plugin directory.

== Installation ==

1. Upload `attendance-list` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [attendancelist] in your post content

== Frequently Asked Questions ==

= Can i have multiple lists in one post? =

No, you can add only one list per post.

= Is there admin panel for list managing? =

No, there isn't, but it will appear in future relases.
